# Roadmap

## Phase 2 — Reader accounts & follows
- Watchlist/follow for games (placeholder button already on game profiles)
- Release-date reminders (email/push) built on the followed-games graph
- Saved searches and followed tags/authors

## Phase 3 — Search & discovery upgrades
- Meilisearch/Typesense behind the existing `/api/search` contract (typo tolerance, ranking, facets)
- Trending signals from real analytics + Twitch viewership (adapter already present)
- Personalized "For you" rail driven by follows

## Phase 4 — Editorial power tools
- CMS draft preview links into the site (secret scaffolded)
- EE review workflows for hard field-level fact-check/SEO gates
- Leads dashboard fed by the RSS adapter; assignment tracking
- Newsletter composer + ESP integration with double opt-in

## Phase 5 — Media & community
- Licensed video ingestion (YouTube provider fields already modeled) with VideoObject upgrades
- Image focal-point cropping + blurhash placeholders
- Comments/reactions with moderation tooling
- Live-blog article type for events

## Continuous
- Localization (hreflang groundwork in SEO layer)
- Web vitals monitoring (RUM) and error tracking
- Accessibility audits each release
